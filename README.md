# CryptoAladdin - Data Pipeline

Модуль 1 системы автоматической ребалансировки криптопортфеля.

## Описание

Этот модуль отвечает за сбор, фильтрацию и обработку данных о криптовалютах с капитализацией более $1 млрд. Система ежедневно обновляет данные и рассчитывает ключевые метрики для последующего анализа.

## Функциональность

- Автоматический сбор рыночных данных с CoinGecko API
- Фильтрация активов по капитализации (> $1B) и ликвидности
- Получение исторических данных (90 дней)
- Расчет метрик: волатильность, доходность, корреляция с BTC, бета-коэффициент
- Сохранение всех данных в SQLite базу
- Генерация отчетов о выполненной работе
